# API Endpoints

## HTML API

# Root

  * `GET /` - Loads React app

## JSON API

# Users

  * `POST /api/users`

# Session

  * `POST /api/session`
  * `DELETE /api/session`

# Artists

  * `GET /artists`
    * For search
  * `GET /artists/:id`
  * `POST /artists/:id`
  * `PATCH /artists/:id`

# Albums

  * `GET /albums`
    * For search
  * `GET /albums/:id`
  * `POST /albums/:id`
  * `PATCH /albums/:id`
  * `DELETE /albums/:id`

# Tracks

* `GET /tracks`
* `GET /tracks/:id`
* `POST /tracks/:id`
* `PATCH /tracks/:id`
* `DELETE /tracks/:id`
