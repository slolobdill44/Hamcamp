# API Endpoints

## HTML API

# Root

  * `GET /` - Loads React app

## JSON API

# Users/Artists

  * `POST /api/artists`
  * `GET api/artists`
    * For search
  * `GET api/artists/:id`
  * `POST api/artists/:id`
  * `PATCH api/artists/:id`

# Session

  * `POST /api/session`
  * `DELETE /api/session`

# Albums

  * `GET api/albums`
    * For search
  * `GET api/albums/:id`
  * `POST api/albums/:id`
  * `PATCH api/albums/:id`
  * `DELETE api/albums/:id`

# Tracks

  * `GET api/tracks`
  * `GET api/tracks/:id`
  * `POST api/tracks/:id`
  * `PATCH api/tracks/:id`
  * `DELETE api/tracks/:id`
